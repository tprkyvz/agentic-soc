
# `research` – Papers, Notes, and Literature Review

Use this directory to organize all **theoretical and background material**
supporting your graduation project.

Typical contents:

- PDF copies of academic papers (if allowed by license).
- Markdown notes summarizing key ideas.
- Comparisons between different approaches or tools.

## Suggested Structure

```text
research/
  reading-list.md        # List of papers with short summaries
  notes/
    mitre-attack.md
    soc-workflows.md
    llm-security-uses.md
```

Keeping your research material here makes it easier to:

- Justify design decisions in `docs/architecture`.
- Trace how your understanding evolved over time.
