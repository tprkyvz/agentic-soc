
# `docs/reports` – Progress and Status Reports

This folder is intended for **time-based reports** that track your progress
throughout the project (approximately one year).

Examples:

- Monthly reports (e.g., `2026-04-report.md`, `2026-05-report.md`).
- Milestone summaries (e.g., `milestone-1-research-completed.md`).

Each report can briefly cover:

- What you planned.
- What you actually did.
- Problems / blockers.
- Next steps for the following period.

Maintaining these reports will also help when writing your final thesis.
