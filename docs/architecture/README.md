
# `docs/architecture` – System Architecture and Design

Use this directory to document the **technical design** of the Agentic SOC.

Typical contents:

- High-level architecture diagrams (PNG/SVG + source files if using tools like Draw.io).
- Sequence diagrams showing agent interactions.
- Data flow diagrams for log ingestion and knowledge base updates.
- Design decisions and trade-offs (e.g., why a certain database or LLM provider).

## Suggested Files

- `architecture-overview.md` – Big-picture view of the system.
- `agent-interaction-sequences.md` – How agents collaborate on a case.
- `data-models-and-storage.md` – Main entities and their relationships.
- `deployment-and-lab-setup.md` – How components run together in the lab.

Keeping this directory well maintained will greatly simplify writing your thesis
and explaining the system to others.
