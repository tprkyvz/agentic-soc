
# `notebooks` – Prototyping and Experiments

This directory is dedicated to **Jupyter notebooks** used for:

- Early experiments with log parsing and feature extraction.
- Testing different LLM prompts and configurations.
- Benchmarking RAG / retrieval strategies and embeddings.

Keep notebooks here instead of inside `src/` to clearly separate
research / exploration code from the main application codebase.

## Suggested Conventions

- Use descriptive names, such as:

  - `01-log-format-exploration.ipynb`
  - `02-llm-prompt-experiments.ipynb`
  - `03-rag-prototype.ipynb`

- At the top of each notebook, briefly document:
  - The **goal** of the experiment.
  - The **date** and **data source** used.
  - Any **key findings** or next steps.
